# Job Application Tracker & Profile Builder

A full-stack-ready job application tracker built with React and scalable state management using useReducer + Context.

## Features
- Multi-step profile form
- Centralized state management
- Validation per section
- Review & submission flow
- Backend-ready architecture

## Tech Stack
- React
- Context API
- useReducer
- Vite
